movieSchema

The following file contains the schema for the movie and user collections in the database. it also contains the models for the movie and user collections. The movie schema contains the following fields: Title, Description, Genre, Director, Actors, ImagePath and Featured. The user schema contains the following fields: Username, Password, Email, Birthday and FavouriteMovies. The user schema also contains a static method to hash the password entered by the user and a method to validate the password entered by the user. The user schema also contains a reference to the movie collection. The movie schema contains a reference to the user collection. The strings are imported from the mongoose module.
